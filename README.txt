== Description ==

#### Loftysms for WordPress

*Sending bulk messages on your wordpress website should be easy and with Loftysms, it finally is with the Loftysms plugin

#### Some of the Loftysms for WordPress features

- Connect with your Loftysms.com account in seconds.
- Check Balance, Make Online Payment etc.
- Send Single SMS, BulkSMS.
- Check Message Delivery Status.
- Create and Manage Contacts.
- Fast and Reliable.
- Saves you time, resources and efforts.
- Highly cost effective, the plugin is available freely and you only need to purchase SMS

== Installation ==

1. In your WordPress admin panel, go to *Plugins > New Plugin*, search for **Loftysms** and click "*Install now*"
2. Alternatively, download the plugin and upload the contents of `loftysms.zip` to your plugins directory, which usually is `/wp-content/plugins/`.
3. Activate the plugin
4. Set [your Loftysms.com personal access token](https://www.loftysms.com/user/apps) in the plugin settings.
5. Click the SendSMS page to send sms at any time
